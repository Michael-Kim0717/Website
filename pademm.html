Home Screen :

This screen is currently incomplete, but my plan for this screen is to ensure that the user is able to instantly see which unit they require once they open the application.

* Details will follow once the screen is completed. *

Monster List Screen :

This screen is reachable from the 'Home Screen' by clicking the 'Monster Box' button.

On this screen, it shows all the units added by the user in the 'Monster Directory' page.

Each listing on the list has the following information respectively : the unit added, the unit that the user wants to obtain, and the user's priority in wanting to evolve this unit.

Every new listing will have the 'unit that the user wants to obtain' as a '?' symbol since no prior details were filled out.

Monster Details Screen :

The 'Monster Details Screen' is shown after clicking on a listing in the 'Monster List Screen'.

Here we have the unit you currently own in the top image and on the bottom are the units that your unit could possibly evolve into. Upon clicking on any of the bottom units, the required material for evolution will show directly under your unit (as shown, there are 5 materials required to evolve your unit).

If you suddenly believe that your unit should be evolved before others, the priority of the unit is also changeable.

Monster Directory Screen :

This screen is the result of clicking the small circular button on the bottom right of the 'Monster List Screen'.

In order for the user to efficiently find their units, the user is able to input part of the name into the search field or directly enter the unit's ID number.

Once the unit is found, the user has the ability to add the unit into their box.
