Password Journal

Status : UNRELEASED, used for personal purposes, I didn't want to release an application that involved other people's personal information.


Home Screen :

Upon opening the application, the user is shown a list of all their manually entered accounts.

The button in the top left corner is to add another account into the list. The three required fields that need to be entered before an account is added are : the account's name, the user's username, and the user's password.

The button in the top right corner brings the user into the settings menu.

Edit Screen :

The 'Edit Screen' popup is the result of clicking an account in the home screen. Of course to even get to this screen, there is a level of security. A PIN number will be prompted if the user has set one in the settings and the screen for the PIN number is shown below in the 'Settings Screen' section.

Upon correctly entering your PIN number (if applicable), the user will see their account's name, username, and password. They also have the ability to either 'Edit' the information presented or 'Delete' the information completely.
Settings Screen :

At the moment, there is only one setting available for the user to change and that is the 'Set/Edit PIN' setting.

If you have never set your PIN number before, they will just have to enter it twice (one as an initial input, the second as confirmation).

However, if their PIN number has been set before, in order to change it, they will need to enter their current PIN number, and only when the input is correct, enter it twice (one as an initial input, the second as confirmation).
